## Part 3: Internal Linking Strategy

### Link Architecture Goals
1. **Hub-to-spokes** — HomePage links to all major categories
2. **Category clustering** — Related pages link to each other
3. **Content-to-community** — Blog/events link to member opportunities
4. **Deep linking** — Navigate from profiles → projects → teams

### Strategic Internal Links

#### HomePage Should Link To:
```
→ /about (Learn the story)
→ /everything-we-do (See all initiatives)
→ /opportunities (Join us)
→ /projects (See student projects)
→ /teams (Meet student leaders)
→ /blog (Read stories)
→ /events or /paradox/home (Upcoming events)
```

#### AboutPage Should Link To:
```
→ /values or add section on core values (Ownership, Impact, Community)
→ /everything-we-do (Detailed breakdown)
→ /teams (The people behind it)
→ /opportunities (How to join)
→ /projects (Results of values in action)
```

#### ProjectsPage Should Link To:
```
→ /projects/{individual projects}
→ /teams (Teams executing projects)
→ /blog (Stories about project execution)
→ /opportunities (How to join)
→ Each project detail links to:
   → Related projects (same category)
   → Team page (who leads it)
   → Join/volunteer opportunity
```

#### TeamsPage Should Link To:
```
→ /teams/{individual teams}
→ /opportunities (Join a team)
→ /projects (What teams execute)
→ /member/{team leads}
Each team links to:
   → Projects they own
   → Team members
   → Join request
```

#### BlogPage Should Link To:
```
→ /blog/{individual posts}
→ /projects (Case studies of projects)
→ /teams (Team profiles)
→ /opportunities (Call-to-action)
Each post links to:
   → Featured project/team
   → Related posts (same topic)
   → Call-to-action pages
```

#### OpportunitiesPage Should Link To:
```
→ /volunteer (Volunteer handbook)
→ /volunteer/apply (Application)
→ /teams (Available teams)
→ /projects (Project opportunities)
→ /opportunities page links back to:
   → /about (Why join)
   → /blog/stories (Real member stories)
   → FAQ or /support
```

#### ProfilePages Should Link To:
```
User profile links to:
   → Their posts
   → Teams they're in
   → Projects they're leading
   → Their achievements/roles
Profile view links to:
   → Similar members
   → Teams with openings
   → Recommended projects
```

### Cross-Content Link Examples

**In Blog Posts:**
```
When discussing "student leadership" → Link to /about, /opportunities
When discussing "project execution" → Link to /projects, /teams
When mentioning "community values" → Link to relevant team/project
When featuring a student → Link to their profile
```

**In Project Pages:**
```
"This project is led by {Team}" → Link to team page
"Join this project" → Link to /opportunities
"Similar projects" → Link to related projects
"Team members" → Link to member profiles
```

**In Team Pages:**
```
"Projects we execute" → Link to project pages
"Join our team" → Link to /volunteer/apply
"Team members" → Link to member profiles
"Our impact" → Link to project metrics/stories
```
